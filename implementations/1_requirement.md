## Requirements

When a user visits the path "/", the application should display the text "Getting started." like this:

localhost:4567

Getting started.

## Implementation

* Create Gemfile
  * add ruby version
  * add gem sinatra
  * add gem sinatra-contrib

* Run Bundle Install

* Create cms.rb
  * require sinatra
  * require sintara-contrib
  * add get "/" route to application.rb
    * add string "Getting started."
